# NutriVisionAi

A smart nutrition analysis application powered by AI (frontend only atm).

## Description

NutriVisionAi uses artificial intelligence to analyze food and provide nutritional information.

## Features

- AI-powered food recognition
- Nutritional analysis
- Visual food detection

## Installation

```bash
git clone https://github.com/yourusername/NutriVisionAi.git
cd NutriVisionAi
```
